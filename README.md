# Kratos
A repo that facilitates the installation and customization of debian linux, and many aspects of the distro with the help of custom scripts.

## Kalix (kratos.sh)
Kalix bootstraps a Debian/Kali environment by installing core packages, setting up Zsh with Oh-My-Zsh and the kali-like twoline prompt, configuring tmux with logging and clipboard integration, and enabling zoxide plus productivity plugins.
